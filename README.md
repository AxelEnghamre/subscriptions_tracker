## Getting Started

- create a .env.locale file or add env variables from the [.env.example](.env.example) file
- run the development server:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000)
- run the builder / compiler
    ```bash
    npm run build
    ```
    Open [http://localhost:3000](http://localhost:3000)
